# ChessMorph - The Adaptive Chess AI

ChessMorph is an adaptive chess application designed to maximize player enjoyment. It morphs its playing style dynamically to keep you in the "Flow State."

## Features

- **Rubber Band AI**: Plays harder when you're winning, and makes "human" mistakes when you're losing.
- **Volatility Injection**: Prioritizes exciting, tactical lines.

## Tech Stack

- **Frontend**: React + react-chessboard
- **Backend**: Python FastAPI + python-chess
- **Engine**: Stockfish
